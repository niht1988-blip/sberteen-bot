# SberTeen Telegram Bot

Telegram-бот для образовательного проекта SberTeen — расписание, запись на мероприятия, сдача ДЗ, сберкоины.

## Установка

```bash
cd sberteen-bot
pip install -r requirements.txt
```

## Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен
2. Скопируйте `.env.example` в `.env` и заполните:
   ```
   BOT_TOKEN=ваш_токен
   ADMIN_PASSWORD=пароль_админа
   ADMIN_IDS=ваш_telegram_id
   ```

## Запуск

```bash
# Заполнить расписание из файла
python seed_data.py

# Запустить бота
python bot.py
```

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Регистрация и главное меню |
| `/admin` | Панель администратора (после ввода пароля) |

## Функционал

### Участник
- 📅 Просмотр расписания с навигацией по страницам
- ✅ Запись / ❌ отмена записи на мероприятия
- 👤 Личный кабинет: запланированные занятия, баланс
- 📝 Сдача ДЗ (фото/документ) на проверку
- 💰 Накопление сберкоинов за одобренные ДЗ

### Администратор
- 📅 CRUD мероприятий (добавление, редактирование, удаление)
- 👥 Просмотр участников, экспорт списка в CSV
- 📝 Проверка ДЗ (очередь pending-заданий)
- 💰 Начисление / штраф сберкоинов
